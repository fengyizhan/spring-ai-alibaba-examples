# Spring AI Alibaba DashScope 翻译示例

[English](./README-en.md) | 中文

## 介绍

演示 Spring AI Alibaba 使用 DashScope 模型实现文本翻译功能。

## 功能特点

- 支持多语言之间的文本翻译
- 提供简单翻译、流式翻译和自定义参数翻译接口
- 易于集成和扩展
